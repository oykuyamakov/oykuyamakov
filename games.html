    • Merge Barista

      Merge Barista is a mobile merge game available on iOS and Android platforms published by Rollic Games.

      I contributed to this project as a Game Programmer at Zerosum, working with an exceptional team over the course of a year. My key responsibilities included:

      • Addressables System Implementation: Implemented Unity's Addressables system to efficiently manage and deliver large-scale data updates and assets, optimizing performance and scalability.
      • Live Operations: Supported live game updates, ensuring smooth integration of new content, features, and bug fixes to enhance player satisfaction.
      • System Architecture: Designed and developed modular systems for managing game state, dynamic content, and progression logic.

      The project was successfully acquired by Rollic, a subsidiary of Zynga, marking a significant milestone for the team.

    • Belgrad Park is a real-time 3D adventure/horror game take place in upstate New York.
